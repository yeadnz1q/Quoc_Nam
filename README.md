Описание приложения

**Currency Converter** — это удобное desktop-приложение на Python для мгновенной конвертации валют с использованием реальных курсов. Приложение поддерживает более 160 валют мира, сохраняет историю операций и работает полностью офлайн после загрузки курсов.

Основные возможности

| Функция | Описание |
|---------|----------|
|  Поиск валют | Выпадающие списки с автодополнением |
|  Конвертация | Мгновенный расчёт по актуальному курсу |
|  История операций | Таблица с деталями каждой конвертации |
|  Сохранение данных | Автосохранение истории в JSON |
|  Обновление курсов | Кнопка для загрузки свежих данных |
|  Валидация ввода | Защита от некорректных данных |
|  Современный интерфейс | Тема `clam` для приятного вида |

  Для кого это приложение?
